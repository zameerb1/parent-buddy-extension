SafeTab Privacy Policy

Last updated: January 2026

What SafeTab Does

SafeTab is a parental control extension that helps parents manage and filter their children's web browsing activity. It is designed to be installed by a parent or guardian on a child's device.

Data We Collect

SafeTab collects the following data and sends it to a private backend server controlled by the parent:

  • Page URLs and titles — to determine whether a website should be allowed or blocked
  • YouTube video metadata — video title, channel name, and a short preview of the video description (first 200 characters) to classify content appropriateness

Data We Do NOT Collect

  • Passwords or form inputs
  • Personal information (name, email, etc.)
  • Cookies or browsing history beyond page titles
  • Full video descriptions, comments, or transcripts
  • Screenshots or screen recordings

How Data Is Used

Collected data is sent to a private server operated by the parent/guardian. It is used solely to:

  • Determine whether to allow or block a webpage
  • Classify YouTube videos as appropriate or inappropriate for children
  • Provide browsing activity summaries to the parent

Third-Party Services

The backend server may use OpenAI's API to classify YouTube video content. Only minimal video metadata (title, channel name, description preview) is sent for classification. No personal data is shared.

Data Sharing

We do not sell, share, or distribute any collected data to third parties. Data is only accessible by the parent/guardian who operates the backend server.

Data Storage

Data is stored on the parent's private backend server. The extension itself stores only the backend server URL and device identifier locally using Chrome's storage API.

Contact

For questions about this privacy policy, please open an issue on our GitHub repository.
