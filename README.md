# Parent Buddy - Chrome Extension

AI-powered parental control extension for Chromebook.

## Installation on Chromebook

### Option 1: Download ZIP
1. Click the green **Code** button above
2. Click **Download ZIP**
3. Extract the ZIP file

### Option 2: Clone
```bash
git clone https://github.com/zameerb1/parent-buddy-extension.git
```

## Load in Chrome

1. Open Chrome and go to `chrome://extensions/`
2. Enable **Developer mode** (toggle in top right)
3. Click **Load unpacked**
4. Select the extracted/cloned folder
5. The extension icon will appear in your toolbar

## Configure

1. Click the Parent Buddy extension icon
2. Enter your backend URL: `https://parent-buddy-production.up.railway.app`
3. Enter Device ID: `default`
4. Click **Save**

## Features

- Blocks internet access when disabled by parent
- AI-powered YouTube video classification
- Parent controls via Telegram bot (@Parentbuddybot)

## Telegram Commands

Message the bot with:
- "allow internet for 30 mins"
- "block internet"
- "allow MrBeast channel"
- "what's being watched?"
- "status"
